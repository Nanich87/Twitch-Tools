About Twitch Tools

Purpose of the project

The main goal of Twitch Tools is to provide free tools and free services that help Twitch users maximize their experience while watching live streams from Twitch. Twitch Tools are designed as a third party website and we are not paired with Twitch.

Project details

Twitch Tools is a Single Page Application (SPA) written in JavaScript, using JQuery for DOM manipulations and AJAX requests, and Sammy for routing.
